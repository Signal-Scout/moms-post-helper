# Mom's Post Helper

A simple tool for posting Amazon affiliate links to multiple Facebook accounts using Claude in Chrome.

## How to use

1. Open the tool in Chrome
2. Click ⚙️ and add your 4 Facebook account names (exact match to Facebook's switcher)
3. Paste your product link and write a rough caption
4. Click Preview, then Copy & Launch
5. Paste the prompt into the Claude sidebar — Claude writes 4 captions, you approve, then it posts automatically

## Setup (one time)

1. Install [Claude in Chrome](https://chromewebstore.google.com/detail/claude/kcgbniefejbpkmbhkfhjohkjohnmfbaf)
2. Sign in with your Claude account
3. Bookmark this page

## Updating

Any changes Jackson makes will appear automatically next time you open the page.
